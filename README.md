# sas_internshala

A new Flutter project.
=======
The application was developed using Flutter and Dart. 
State management is implemented using Provider. 
Story narration is handled through flutter_tts, while quiz rendering is fully data-driven from JSON.
The application follows an MVVM-inspired architecture with separate Models, Providers, Screens, and Widgets to ensure scalability and maintainability.


============================
We used the Flutter TTS completion callback along with Provider state management for managing the transition state between audio ending and quiz appearing.


======================================

Quiz is fully data driven, means Questions ,options,answers etc are parsed from JSON Models and UI will dynamically showing everything.


=========================================


The app transitions through loading, playing, quiz, success, and error states by using provider statemanagement.
